## 一、本周概述

本周完成 F1TENTH 两个月落地方案中 **W5–W8** 阶段的软件交付，即创新点 4（特权专家 → BC/DAgger 学生）

### 1. 明确 W5–W8 设计与诚实约束

本周首先把方案的「加分层」收敛成一个自包含的离线蒸馏包，并明确两条诚实约束写进设计与论文。

| 设计点 | 结论 |
| ---- | ---- |
| 包结构 | 单一 `src/slash_distill`（离线 `python -m` + ROS2 部署节点 + launch） |
| 特权专家 | 中心线 pure-pursuit + g-g-v 限速（无中心线时回退反应式专家） |
| 学生 | 无状态帧堆叠 1D-CNN（层数无关），输入多层 lidar + 速度 + opponent_state |

| 诚实约束 1 | f110_gym 是 2D，「3D 多层感知」收益**无法在仿真内度量**，该消融为实车/定性 |
| 诚实约束 2 | 横向 `ay_max` 仍为假设值（W2 未补绕圆 bag），接地仿真标 `ASSUMED` |

设计写入 `F1TENTH_W5-W8_蒸馏部署_设计spec.md`。

## 四、问题与风险

| 问题 | 现象 | 根因 | 影响 | 当前应对 |
| ---- | ---- | ---- | ---- | ---- |
| 2D 仿真无高度 | 「3D 多层」收益无法在仿真度量 | f110_gym 是 2D | 感知维度消融不能在仿真出 | 设计为实车/定性消融，学生网络层数无关，诚实写进论文 |
| 横向 `ay_max` 仍假设 | 接地仿真横向极限标 ASSUMED | W2 未补绕圆 bag | sim 横向略保守 | 绕圆 bag 实测后改 `sysid_params.yaml` 即真接地 |
| 接地是命令侧近似 | gym 单轨 + 命令侧 K/τ/Td | 模型族不同 | sim-to-real 仍有残差 | 论文据此写参数失配；以实车低速复核 |
| 无 GPU 环境训练慢 | 沙箱 CUDA 不可用 | 离线环境限制 | 仅 smoke 验证 | 正式训练在 Jetson GPU / 工作站跑 |
| onnxruntime 未装 | 数值自检/节点 onnx 后端暂缺 | aarch64 镜像下载慢 | 不影响导出与反应式兜底 | 按 W7 指南装 NVIDIA Jetson onnxruntime-gpu |

## 六、本周结论

本周完成了 W5–W8 的全部软件交付：在 sysid 接地的 f110_gym 中，用中心线 pure-pursuit 特权专家产示范，

已成立的「3D 反应式 + sysid + 安全壳」主干。下一步重点不是继续堆功能，而是放入真实赛道、
在 GPU 上正式训练、在 Jetson 上构建 TensorRT 引擎并完成实车单车低速复核与消融实证。
